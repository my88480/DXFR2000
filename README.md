# DXFR2000
Operate AutoCAD DXF file without AutoCAD environment. compatible with AutoCAD R2000 at least.

ENTITIES:<BR>
----ARC<BR>
----CIRCLE<BR>
----LINE<BR>
----LWPOLYLINE<BR>
----POINT<BR>
----TEXT<BR>
----ELLIPSE<BR>

Now, this project support chinese character set(use Arial.ttf truetype font). Thanks https://github.com/JadeChong <BR>
